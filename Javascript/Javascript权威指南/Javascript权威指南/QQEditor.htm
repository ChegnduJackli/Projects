前言
   一直都想找个时间来好好总结这半年的生活，然而总没有合适的机会，有时又没有灵感，面对着电脑，脑壳却一片空白， 不得不承认，某些创造性的基因有了些许退化。However,还是大概回顾总结这半年的生活。
生活
  对于上班族的生活，大多时候算是平淡而稳定的，平淡是一种幸福，其实一点都没有错。一次跟同事们吃饭， 同事说了句话：“其实现在不是挺好的吗？不愁饭吃，不愁没有衣服穿，有份工作，有几个朋友，干嘛还把自己搞那么累”。 听了，我也觉得很实在，不愁吃，不愁穿，可是，生活现实是这样的吗？不是的。为什么不是，因为我们不知足！
  有时有人问你工作几年了，回答是3年，是的，3年，好像很快，还清晰记得大学毕业2012年6月30号拍毕业照的场景。 然后回头一想，经历过了那么多事，走过了那么多弯路。3年，其实是差不多的，毕竟来SG,都超过一年半了。一年半，快吗？ 其实是差不多的，从刚来的时候，什么英语完全听不懂，听力从来都是高中拖后腿的题目，从来都是猜，有一种压力叫无处发泄。
   吐槽下这边的Bus,这边的Bus是不会报站的，也没有电子屏幕显示，那个时候手机买的国产魅族，国外不支持GPS导航，怎么办，猜吧，于是，经常下错站。 终于，狠下心来，换个手机吧，从此也不用担心迷路了 。Bus是全程密封，毕竟开的空调，但是不晓得为嘛，晕车，味道难闻，让我对bus望却止步。
   然而地铁，确实是非常方便的，出门也从来优先选择地铁，到什么地方都很快，毕竟城市小。但是，它为嘛总是出故障。听说昨天还停到隧道了，没电，没空调，今天还维修来着。
   很欣慰的是，这个Team的人都比较不错，工作一起吧，很和谐。闲暇之余还可以一起玩，比如打打球，玩玩游戏。 从一个新同事来之后，台球，就开始带动了起来，一般每周五中午吃完饭，去打一个来小时。但是有时想玩了，就叫上一起去，记得上个星期，周一到周五，每天中午都去， 真感觉是走火入魔了。有时周末，就约着打乒乓球，骑车，游泳。
  所以整体说来，不算枯燥，时常出去玩，我觉得很满足。
健康
  人在生病的时候，就会发现，有个健康的身体是多么的重要，半年来，生病2次，有一次估计是冷热交替，有次居然是坐木乃伊过山车给吓的。 对于我们这种长期坐在办公司做事的，通常都会有职业病，如果不注意，就会恶化，听起来很吓人吧，其实真的也是蛮吓人，我们应该每周保持适当的运动，徒步，游泳，骑车，打球等。 身体是1，然后才能加0，话俗理不俗。在身体投资上，绝对不可以疏忽，病到了，一切都没有了。
工作
  最近同事离职，新人还没有进来，变得略微忙碌。然后8月份另一个同事辞职，一个同事回中国，年底，一个同事申请换项目组， 整个Team都换差不多了，只是本来中国同事就不多，这下都走了 ，我这可以聊天的对象又少了，只是那个时候，我还在不在这个Team,只有到时候再说了。 他们在这个Team都是呆了4年或者以上，然而今年那么大的调动变化。却也是始料未及的。工作就是这样，旧人走，新人来，然后继续着，重新融合一个团队。一个和谐的Team，真的是可遇不可求的。 曾经是，现在是，就不知道未来是不是。
   做维护性项目，一般不会用多么高深的技术，但是依旧还是会遇到从新接触的东西，然后你从中学习，汲取经验。所以整体水平还是有提高。 时常做些无偿的加班在所难免，庆幸的是只是偶尔。要想学习新东西，从来都是靠自己，腾出时间去学习，去充电。这样才能逐渐的成长。知道的越多，路就走得越远。 整体说来，工作算很平稳。
旅游
   这半年，去了植物园，动物园，鸟园，每样都有代表性，都有不同的特色。最庆幸的是办了张环球影城的半年票， 这样，一个月都会去一两次。半年，应该差不多10次了。上周挑战大过山车，然后坐了小的过山车，总共八遍，却没有厌倦。
  由于以前在成都的国色天香玩，大摆锤。那个时候不知道天高地厚，什么都不怕，越刺激越要去尝试， 于是就去玩大摆锤，坐上去之后，当然失重，超重，旋转，少不了，开始的时候还能候出来，后面感觉实在不行了，已经吼不出来了，然后就憋着， 难受到不行，但是它还在继续使劲摇摆。终于下来了之后，就头晕目眩，脸色苍白，休息了半个小时，才缓过神来。然后后面稍微刺激点项目，我都不去了。 从那个时候，我就对这种失重超重厉害的项目，心里感到恐惧。就产生了阴影，说来也是弱爆了，估计当时是憋着了，玩这种项目，是不能憋的啊，可是已经吼不出声了。
   记得过年去广州玩的时候，去广州塔，塔顶大概430米高，上面有娱乐项目，有摩天轮（在顶层建筑边缘），有冲上云霄（很高的地方落下来）。本来是玩摩天轮的，但是排队2个小时后， 发现还遥遥无期，工作人员开始疏散，可以替换成其他的项目，就去挑战了冲上云霄。做了多次心里斗争，才坐上去，上升到30米的时候，其实是（430+30米），我的天，怎么那么高啊， 能看到地面的汽车像蚂蚁样。原来我也还是有点恐高。全程闭着眼睛。下来后，发现还是蛮刺激，可以接受，失重，没有超重难受。冲上云霄有2个，一个速度快点，一个慢点，挑战了个慢点的后。 就灰溜溜的走了。毕竟快的那个，太吓人了。
   5月份，这边的双规过山车恢复后，就没有胆量去挑战，去了2次，就在下面望望，就溜走了，经常被同事调侃。第三次去的时候，鼓足勇气去挑战了下，从排队到上车的过程，全程紧张，害怕。 但快上车的时候，真的差点就放弃了，还好旁边的同事，鼓励着，就硬着头皮上。坐到倒数第二排，上都顶，开始下降的时候，我的天，看不到下面，有段下降的距离是垂直的，好吧，眯着眼睛，一秒的失重， 然后就没有失重，就正常了。就是速度快了些。下来之后，才发现，好像除了感受到失重，什么感觉都没有。但是，觉得又成长了。
情感
  这一两年来，空间秀最多是就是谁谁结婚，谁谁生孩子，谁又在哪儿秀孩子。应该恭喜他们，虽然拉了仇恨， 但是他们都找到了自己一辈子的伴。
  天天坐在一个办公室的同事，突然说他要结婚了。记得是元旦回去有交个女朋友，对于突然提出的要结婚，我依旧表示震惊。 我问：“怎么突然就想结婚了？”，他答：“因为遇到个合适的了，没有必要再等了。”我觉得这个回答很有代表性，也很有力度。
  对于我，我只想说，得之我幸，失之我命，随缘，如此而已。
总结
  社会是复杂的，生活是简单的，如何让简单的生活过得精彩而有意义，一切在于心。
  路漫漫其修远兮，吾将上下而求索。
  脚踏实地，仰望心空。
  年轻人，努力，奋斗，没有借口。
将来
  愿健康，平安，顺利，更多惊喜。
